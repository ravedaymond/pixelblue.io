Typewriter styled text animations for any webpage using HTML5, CSS3, JavaScript, and jQuery.

Overview

This lab is a script written in JavaScript using the jQuery library. I wanted a way to animate text with a typewriter effect for this webpage as seen on the initial page load, and this script was a great way of brushing up on my programming and learning a little more about the connections between HTML5, CSS3, and JavaScript (jQuery).

I figured others may be looking for a similar script, and I plan to work on making it more readily ‘plug-n-play’ for anyone to use in their projects.

Full documentation is available on GitHub, along with the source file documented in JSDoc notation. You can also view it live using the Pen below.
